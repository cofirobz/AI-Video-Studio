# assets/

Shared brand assets, reused across episodes, in subfolders:
- `characters/` — generated reference sheets and per-character asset packs
- `backgrounds/` — reusable generated backgrounds/settings
- `logos/` — channel/brand logos and watermarks
- `fonts/` — brand fonts for captions, titles, thumbnails

Assets generated *for one specific episode* (its scene images/video/voice) live in that episode's own `episodes/{id}/assets/` instead — see `docs/architecture.md`.
